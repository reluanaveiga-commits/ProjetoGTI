 Floresta - Jogo 2D em Pygame

 Sobre o projeto

Floresta** é um jogo 2D desenvolvido em Python utilizando a biblioteca **Pygame**.

O jogador controla um personagem em um ambiente de floresta, enfrentando inimigos, desviando de obstáculos e acumulando pontos durante as fases.

O projeto foi desenvolvido com foco em programação orientada a objetos, organização de código e criação de uma estrutura de jogo utilizando entidades,
colisões, fases e gerenciamento de recursos.


Funcionalidades

 Menu inicial
 Sistema de fases
 Sistema de pontuação
 Ranking Top 10
 Controle de vida do jogador
 Inimigos com movimentação e ataque
 Sistema de tiros
 Obstáculos da floresta
 Colisão entre entidades
 Música durante as fases
 Diferentes modos de jogador
